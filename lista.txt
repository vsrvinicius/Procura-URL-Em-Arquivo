WatchGuard Cloud Public URLs

Notes:
------
* All traffic to WatchGuard Cloud uses TCP port 443, with one exception. 
  Fireboxes that run Fireware v12.2.1 or lower connect on port 8883.
* This list includes the base domain URL, not the full URL path.
* WatchGuard Cloud operates in three regions. Region-specific URLs contain one of these region codes:

      jpn = APAC region
      deu = EMEA region
      usa - NA / Americas region
